# Job Listing Application with Frontend UI in ReactJS and Backend Using Spring Boot & MongoDB

Job listing application with possibility to browse through all offers, search for offers by keywords, or add new offers if you are an employer. The platform is implemented using ReactJS for the frontend, which includes a welcome page, a page to create new offers and a page to list the offers. The backend is implemented in Java using Spring Boot and connecting to the database with MongoDB.
